Chapter 3 Creating a Country Level Map with Sampling Points

In this first exercise we will use rnaturalearth, simple features and ggplot2 to create a map of Brazil and states within Brazil to use as the base of the maps we will create in these exercises.

3.1 Get the Data to Create Our Map

3.1.1 Add a Simple Features file of Brazil

This is our base layer, Brazil, of the map from (Naturalearth.com)[https://naturalearth.com/] using the base plot() to display these data.

library(rnaturalearth)

br_sf <- ne_states(geounit = "brazil",
                   returnclass = "sf")

plot(br_sf)
## Warning: plotting the first 9 out of 83 attributes; use max.plot = 83 to
## plot all

3.2 Plot using ggplot2

Using ggplot2, we have more control over how the data are displayed.

First plot the whole country using ggplot().

library(ggplot2)

br <- ggplot(br_sf) +
  geom_sf(fill = "white")

br

3.2.1 Add Labels to the States

br +
  geom_text(
    data = br_sf,
    aes(x = longitude,
        y = latitude,
        label = abbrev),
    size = 2.5,
    hjust = 1
  )

br

3.2.2 Final Touches

Label the x and y-axis and turn off the legend. The ggplot2 theme, theme_bw() is nice to use with maps, so we will apply that here as well.

br +
  xlab("Longitude") +
  ylab("Latitude") +
  theme(legend.position = "none") +
  theme_bw()
